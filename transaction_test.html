SuperPaymaster Evaluation: Workflow Comparison

Traditional Workflow

Acquire native token (e.g., ETH) from an exchange.
Complete KYC and wait for withdrawal clearance.
Create and set up a standard EOA wallet.
Transfer tokens from exchange to wallet, paying a fee.
If needed, bridge tokens to the correct L2 network, paying more fees.
Initiate transaction in DApp.
Manually approve gas fees and the final transaction in wallet.
